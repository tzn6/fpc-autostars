# AutoStars — плагин автовыдачи Telegram Stars для FunPayCardinal

Плагин для [FunPayCardinal](https://github.com/sidor0912/FunPayCardinal), который **автоматически продаёт и выдаёт Telegram Stars**.

Когда покупатель оплачивает на FunPay лот со звёздами «по username», плагин сам:
1. ловит новый заказ;
2. определяет количество звёзд и Telegram-юзернейм получателя;
3. проверяет юзернейм через **Fragment**;
4. покупает звёзды на Fragment и оплачивает их **с твоего TON-кошелька** (Wallet V5R1 → tonapi.io);
5. дожидается подтверждения транзакции и пишет покупателю «готово».

Бонусом — **авто-ответ на отзывы** покупателей.

Всё настраивается **прямо в Telegram-боте FPC**, без правки файлов.

> ⚠️ **Плагин работает с реальными деньгами (TON) и приватными данными** (сид-фраза кошелька, Fragment cookies).
> Используй **отдельный** кошелёк под автовыдачу и держи на нём только рабочий баланс. Автор ответственности не несёт.

## Установка

1. Скачай **`autostars.py`** и положи его в папку `plugins/` твоего FunPayCardinal.
2. Установи зависимость для работы с TON (в venv, где запускается FPC):
   ```bash
   /путь/к/FunPayCardinal/venv/bin/pip install pytoniq
   ```
   (`requests` уже идёт вместе с FPC).
3. **Активируй плагин** в ТГ-боте: `🔌 Плагины → AutoStars → Активировать`.
4. **Перезапусти FunPayCardinal** (хендлеры и страница настроек навешиваются при старте).
5. В логах должно появиться:
   ```
   [AUTOSTARS] Сервис запущен.
   [AUTOSTARS] Страница настроек в Telegram зарегистрирована.
   ```

## Настройка (в Telegram-боте)

`🔌 Плагины → AutoStars → ⚙️ Настройки`. Секретные значения скрыты и отображаются как «задано».

| Пункт | Что это |
|-------|---------|
| 🍪 Fragment cookies | Cookie авторизованного аккаунта на fragment.com |
| #️⃣ Fragment hash | Параметр `hash` из запросов к `fragment.com/api` |
| 🔐 Сид-фраза V5R1 | 24 слова кошелька TON (Wallet V5R1), с которого оплачиваются звёзды |
| 🔑 tonapi токен | Токен [tonconsole.com](https://tonconsole.com) (необязательно, но убирает задержки) |
| 👤 Показывать отправителя | Показывать ли тебя как отправителя на Fragment |
| 📢 Реклама в комментарии | Пересобирать коммент к TON-транзакции (по умолчанию выкл — надёжнее) |
| 💸 Возврат при ошибке | Авто-возврат заказа на FunPay, если перевод не удался |
| ⏱ Интервал цикла | Как часто проверять готовые заказы (сек) |
| 🪫 Мин. баланс (TON) | Порог баланса: если ниже — автовыдача приостанавливается. 0 — выключено |
| 🔔 Уведомить о низком балансе | Присылать уведомление в Telegram при паузе/возобновлении |
| 📝 Ответ на отзыв | Авто-ответ на отзывы покупателей (вкл/выкл + текст) |
| 💬 Сообщения покупателю | Тексты ответов покупателю (успех/ошибка/неверный ник и т.д.) |

После заполнения ключей нажми **♻️ Переподключить** — бот ответит адресом кошелька и балансом.

## Автовыключение при низком балансе

Если баланс TON-кошелька падает ниже **«Мин. баланс»**, плагин:

1. Приостанавливает обработку переводов (новые заказы продолжают приниматься и ставятся в очередь).
2. Пишет в Telegram сообщение вида:
   > ⚠️ **AutoStars: низкий баланс!**
   > Текущий баланс: **0.3500 TON**
   > Порог: **1.0 TON**
   > Автовыдача **приостановлена**. Пополните кошелёк — плагин возобновит работу автоматически.
3. После пополнения кошелька плагин **сам снимает паузу** и шлёт уведомление о возобновлении.

Установи значение **0**, чтобы отключить функцию.

> Уведомления отправляются в чат, из которого ты последний раз открывал настройки AutoStars. Зайди в настройки хотя бы раз после установки, чтобы бот запомнил твой чат.

### Где взять Fragment cookies и hash
1. Зайди на **https://fragment.com**, авторизуйся через Telegram и подключи TON-кошелёк (тот же, чью сид-фразу вписал).
2. Открой `https://fragment.com/stars/buy`, нажми **F12 → Network**.
3. Введи в поле получателя любой `@username` — появится запрос `api?hash=...`.
4. Кликни по нему:
   - **hash** — значение после `?hash=` в Request URL;
   - **cookies** — строка `Cookie:` целиком из Request Headers.

> Cookies/hash со временем протухают — если переводы перестали идти с ошибкой Fragment, обнови их тем же способом.

### Где взять сид-фразу
Поставь **Tonkeeper** (создаёт кошелёк V5R1), заведи **отдельный** кошелёк под автовыдачу, пополни TON, экспортируй 24 слова: Настройки → Recovery phrase. Вставляй слова через пробел.

### Где взять токен tonapi
[tonconsole.com](https://tonconsole.com) → войти → tonapi → **API keys → Create** (тип **Unlimited**, для backend) → скопировать токен.

## Лоты на FunPay

Плагин ловит заказы из категории **Telegram → Звёзды**, тип **«По username»**. Покупатель указывает свой `@username` при оформлении или присылает в чат:
```
/stars свой_username
```
Можно с `@` и без — без разницы.

## Команды покупателя

| Команда | Действие |
|---------|----------|
| `/stars username` | Указать / исправить Telegram-юзернейм для текущего заказа |

- `requests` — HTTP к Fragment и tonapi.io (уже есть в FunPayCardinal).
